# 系统英语听说 / 口语课程

这是一套面向零基础中文学习者的英语听说与口语入门课程。课程目标不是让你一开始就背很多单词，而是先建立“听得懂一点、敢开口一点、能组织简单句子”的学习闭环。

## 适合谁

- 英语几乎零基础，或多年没学、想重新开始的人
- 想先提升日常交流、问候、自我介绍和基础听力的人
- 希望用中文说明逐步理解英语发音、重音、语调和意群的人

## 学习方式

每一课都建议按下面顺序学习：

1. **先听**：不用追求每个词都听清，先抓声音的高低、停顿和重复出现的词。
2. **再跟读**：小声模仿，再正常音量说出来；重点是节奏，不是完美口音。
3. **再替换**：把句子里的姓名、地点、时间、感受换成自己的内容。
4. **最后输出**：用 2 到 4 句英语完成一个小场景。

## 给零基础学习者的提醒

- 你不需要一开始就听清所有单词。真实交流中，先抓关键词和语气就已经很有用。
- 你可以先说短句。短句说稳，比复杂句说错更能建立信心。
- 每天 10 到 15 分钟，比偶尔一次学很久更适合听说训练。
