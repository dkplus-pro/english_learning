# Lesson 09: Asking for Help and Handling Problems

## 本课目标

学完本课后，你可以：

- 在听不懂、迷路、设备/订单出问题时求助。
- 用“问题 + 需要 + 感谢”的结构组织语言。
- 练习听问题解决场景中的关键词：problem, help, need, can’t, sorry。

## 1. 求助核心句

| 英语 | 中文 |
| --- | --- |
| Can you help me? | 你能帮我吗？ |
| I need help. | 我需要帮助。 |
| I don’t understand. | 我不明白。 |
| I’m sorry, I’m new here. | 不好意思，我刚来/不熟。 |
| I can’t find my hotel. | 我找不到酒店。 |
| My phone is not working. | 我的手机不能用了。 |

## 2. 说明问题

先用最短结构：

```text
I have a problem.
My ________ is not working.
I can’t ________.
Can you help me?
```

例子：

```text
I have a problem.
My card is not working.
Can you help me?
```

## 3. 场景：遇到问题怎么组织语言

顺序：

1. **礼貌引起注意**：Excuse me. / Sorry.
2. **说问题**：I have a problem. / I can’t ...
3. **说需求**：Can you help me? / Could you check it?
4. **确认下一步**：What should I do?
5. **感谢**：Thank you for your help.

### 模板

```text
Excuse me. I have a problem.
My phone is not working.
Can you help me?
What should I do?
Thank you.
```

## 4. 听力：问题场景听什么

别人回应你时，重点通常是动作建议：

```text
Please wait here.
Please call this number.
You need to go to the front desk.
Try again later.
```

先抓动词：`wait`, `call`, `go`, `try`。

### 是否要听清所有词？

问题解决场景中，安全最重要。听不清下一步时要问：

```text
Sorry, what should I do?
Could you write it down?
Could you speak slowly?
```

## 5. 小对话

```text
A: Excuse me. Can you help me?
B: Sure. What’s the problem?
A: My card is not working.
B: Please try again.
A: It still doesn’t work.
B: Please go to the front desk.
A: Thank you for your help.
```

### 意群拆句

```text
Can you help me?
What’s the problem?
My card / is not working.
Please go / to the front desk.
```

## 6. 练习

### 练习 1：三个问题

```text
My phone is not working.
My card is not working.
I can’t find my hotel.
```

### 练习 2：求慢一点

```text
Could you speak slowly?
Could you write it down?
Could you say that again?
```

### 练习 3：动作关键词

听建议时只写动作：

```text
wait / call / go / try / check
```

## 7. 本课检查

- 能说 `Can you help me?` 和 `I have a problem.`
- 能用 4 句说明一个简单问题。
- 听解决方案时能抓动作词，并会请求重复或写下来。
