# Lesson 10: Opinions and Preferences

## 本课目标

学完本课后，你可以：

- 表达喜欢、不喜欢、偏好和简单原因。
- 把短句扩展成“观点 + 原因 + 例子”。
- 听观点类内容时，抓态度词和原因词。

## 1. 表达喜好

| 英语 | 中文 |
| --- | --- |
| I like it. | 我喜欢它。 |
| I don’t like it. | 我不喜欢它。 |
| I love coffee. | 我很喜欢咖啡。 |
| I prefer tea. | 我更喜欢茶。 |
| It’s good. | 它不错。 |
| It’s too expensive. | 它太贵了。 |

## 2. 给原因

| 英语 | 中文 |
| --- | --- |
| because | 因为 |
| easy | 容易的 |
| difficult | 困难的 |
| interesting | 有趣的 |
| useful | 有用的 |
| comfortable | 舒服的 |

### 句型

```text
I like English because it is useful.
I prefer tea because it is comfortable.
I don’t like this shirt because it is too expensive.
```

## 3. 场景：怎么组织观点

顺序：

1. **先说观点**：I like ... / I prefer ...
2. **给一个原因**：because it is ...
3. **给一个小例子**：I use it every day.
4. **问对方**：What about you?

### 模板

```text
I like English.
It is useful.
I use it at work.
What about you?
```

如果 `because` 不熟，可以先分成短句，不强行说长句。

## 4. 听力：观点类内容听什么

观点类句子常有两个重点：态度和原因。

```text
I like this cafe / because it is quiet.
```

听力记录：

```text
Opinion: like cafe
Reason: quiet
```

常见态度词：

```text
like, love, prefer, don’t like, good, bad, easy, difficult
```

### 不必每个词都清楚

如果你听出 `like` 和 `quiet`，即使没听清 `because it is`，也能理解“喜欢，因为安静”。

## 5. 小对话

```text
A: Do you like coffee?
B: Yes, I like coffee.
A: Why?
B: Because it is good and useful for work.
A: I prefer tea.
B: Why?
A: Because it is comfortable.
```

### 意群拆句

```text
Do you like / coffee?
Because / it is good.
I prefer / tea.
Because / it is comfortable.
```

## 6. 练习

### 练习 1：三种观点

```text
I like ________.
I don’t like ________.
I prefer ________.
```

### 练习 2：加原因

```text
I like ________ because it is ________.
I prefer ________ because it is ________.
```

### 练习 3：听观点笔记

听别人表达观点时，只记两栏：

```text
Opinion: ________
Reason: ________
```

## 7. 本课检查

- 能表达喜欢、不喜欢和更喜欢。
- 能用原因词扩展一句观点。
- 听观点时知道抓态度词和原因词。
